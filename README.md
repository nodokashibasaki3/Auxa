# Auxa App

Auxa is a smart lighting application designed to create personalized sensory environments for individuals with autism and sensory processing differences. By combining user preferences with biometric data from wearable devices, Auxa automatically adjusts lighting to improve comfort, reduce anxiety, and enhance well-being.

## Core Features

**📋 Personalized Light Profiles**
- Complete a simple onboarding survey to create your sensory profile
- Set light sensitivity thresholds and color preferences
- Create custom light modes for different activities and moods

**🔄 Manual Mood Check-in**
- Log your current emotional state with intuitive emoji-based selection
- Receive lighting recommendations based on your mood
- Track mood patterns over time to better understand your sensory needs

**💡 Smart Light Control**
- Control Philips Hue, Google Home, and other compatible smart lighting systems
- Apply pre-configured light modes with a single tap
- Adjust brightness, color temperature, and intensity

**📊 Insights Dashboard**
- View your mood history with simple, visual graphs
- Track which lighting settings work best for you
- Monitor sensory patterns and make informed adjustments

**❤️ AUXA™ Wearable Integration**
- Connect compatible heart rate monitoring wearables
- Enable auto-adjustment of lighting based on physiological stress signals
- Receive gentle notifications when stress levels rise
- Calibrate to your personal baseline for accurate stress detection
